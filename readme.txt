
Данные:
- чековые данные (transactions.parquet (2 gb, поэтому не загружены в репо), для чтения через pandas дополнительно нужно установить библиотеку pyarrow)

- справочник магазинов (plants.csv)
- справочник клиентов (clients.csv)
Более подробное описание данных дано в файле data description.

Цель: 
1) проанализировать данные и определить оптимальную методологию определения отточных клиентов

2) разработать модель вероятности оттока клиентов по выбранной методологии

3) дать интерпретацию разработанной модели, ответить на вопросы: какие признаки наиболее влияют на отток клиентов

Дополнительное задание: написать код алгоритма случайного леса, используя только стандартную библиотеку Python и numpy. Для тестирования используйте датасет boston.
